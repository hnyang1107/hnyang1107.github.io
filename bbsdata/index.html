BBS Forum Dataset for Chinese Universities

An user-based dataset crawled from four Chinese universities: Peking University (PKU), Renmin University of China (RUC), Shanghai University (SHU) and Fudan University (FDU)

Introduction

We use web crawlers to download the data from four university official BBS forums: Wei Ming BBS (http://bbs.pku.edu.cn), Tian Di Ren Da BBS (http://bbs.ruc.edu.cn), Le Hu BBS (http://bbs.shu.edu.cn) and Ri Yue Guang Hua BBS (http://bbs.fdu.edu.cn) and we crawled the data in accordance with these websites’ terms of services. We extract the content in particular HTML tags, including post ID, board, time stamp, replied ID. As for text content in message stream, we remove very common Chinese-language stop words such as yi ge (which means “a/an”in English). In addition, we remove some university-related words such bei da (“Peking University”in Chinese) and specific set of jargon in BBS, which do not help to create meaningful topics.

Data Description

Download the data from: bbsData.zip.

The compressed package contains four universities' csv files, the main fields of each file and the explanations are:

Filed Data Type Explanation
post_id CHAR users' ID of the poster
post_time DATE time of the post
target CHAR ID of the user reply to
board_name CHAR name of the board
thread_title TEXT title of the thread
content TEXT contnet of the post

How to Use the Dataset

Our data file is encoded in UTF-8. If you are using pandas, you can follow the scripts below to read our data:

from pandas import read_csv
df = read_csv('file_name.csv', index_col='post_time', parse_dates=['post_time'], infer_datetime_format=True)
